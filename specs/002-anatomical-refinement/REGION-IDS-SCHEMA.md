## ID Generation Rules

```typescript
// Function to generate region ID
function generateRegionId(
  primaryRegion: string,     // "neck", "shoulder", "arm", etc.
  side: "left" | "right",    // Laterality
  subdivision: string        // "anterior", "superior", "palma", etc.
): string {
  return `${primaryRegion}_${side}_${subdivision}`;
}

// Example usage
const id = generateRegionId("hip", "left", "superior");
// Result: "hip_left_superior"
```
